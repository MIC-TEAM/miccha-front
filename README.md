## ⚙️ 프로젝트 구동 방법

|                | 명령어       |
| -------------- | ------------ |
| 패키지 설치    | yarn install |
| 로컬 서버 실행 | yarn dev     |
| 빌드           | yarn build   |
| 빌드 파일 실행 | yarn start   |
